# Cells and Serpents, a survival game!

### By Sophia Chang, Angela Chen, Briana Kirkman, Amanda Yu

### Install
```bash
pip3 install google-genai
pip3 install python-dotenv
```

### Set Up (for mac):
```bash
python3 -m venv venv
source venv/bin/activate
pip install google-generativeai
python main.py
```
